## STRUCTURE PDF OBLIGATOIRE

### 1. SYNTHÈSE EXÉCUTIVE (1-2 pages)
- Point GPS et périmètre étude précis
- 7 dimensions collectées avec taux de complétude
- Sources principales mobilisées par niveau fiabilité
- Enjeux majeurs identifiés pour transformation territoriale
- Recommandations pour Phase B (export JSON)

### 2. CONTEXTE HYDROLOGIQUE QUANTIFIÉ (2-3 pages)

#### 2.1 Données de base
- **Niveau d'eau** : Valeur NGF ou hauteur relative, période 2022-2025, tendances
- **Débit** : m³/s ou estimation argumentée, variabilité saisonnière
- **Qualité eau** : Indices biologiques (IBD, IBGN), paramètres chimiques récents

#### 2.2 Station de référence
- Code station Banque Hydro la plus proche
- Distance au point d'exploration
- Représentativité des mesures

#### 2.3 Sources et fiabilité
- Banque Hydro : URLs stations + codes
- Agences de l'Eau : points de mesure qualité
- Niveau fiabilité et représentativité géographique

### 3. BIODIVERSITÉ CARACTÉRISTIQUE - 5 ESPÈCES (3-4 pages)

#### 3.1 Espèces végétales (2 obligatoires)
Pour chaque espèce :
- Nom vernaculaire + nom scientifique complet
- Statut conservation Liste Rouge UICN
- Abondance locale et rôle écologique spécifique au site
- Habitat préférentiel et exigences écologiques
- Sources identification (Atlas régional, INPN, terrain)

#### 3.2 Espèces animales (2 obligatoires : 1 terrestre + 1 aquatique)
Même structure que végétales + :
- Période d'observation et saisonnalité
- Position trophique dans écosystème local
- Indicateur qualité écologique (pour espèce aquatique)

#### 3.3 Espèce bio-indicatrice (1 obligatoire)
- Valeur indicatrice précise (qualité eau, écosystème...)
- Seuils de tolérance documentés
- Évolution population 2022-2025 si données disponibles
- Pertinence comme sentinelle écologique

#### 3.4 Validation taxonomique
- Codes taxons INPN/GBIF
- Validation nomenclature scientifique
- Cohérence écologique assemblage d'espèces

### 4. MÉMOIRE TERRITORIALE ET VOCABULAIRE LOCAL (2-3 pages)

#### 4.1 Termes hydrologiques vernaculaires (minimum 2)
- Terme local exact + définition technique précise
- Contexte d'usage traditionnel documenté
- Géolocalisation : commune/canton d'emploi
- Étymologie si disponible

#### 4.2 Phénomènes naturels (minimum 2 expressions)
- Expression locale pour phénomène saisonnier
- Phénomène scientifique correspondant
- Période d'occurrence dans l'année
- Intégration dans culture locale

#### 4.3 Pratiques traditionnelles (minimum 2)
- Technique ancestrale liée au cours d'eau
- Description détaillée savoir-faire
- Période historique d'usage principal
- Statut actuel : vivante/patrimoniale/disparue
- Mode de transmission générationnel

#### 4.4 Sources linguistiques et ethnographiques
- Archives départementales, Atlas linguistiques
- Témoignages oraux validés
- Publications ethnobotaniques/anthropologiques

### 5. EMPREINTES HUMAINES STRATIFIÉES - 3 ÉLÉMENTS (3-4 pages)

#### 5.1 Infrastructure hydraulique majeure (1 obligatoire)
- Nom officiel + gestionnaire
- Description technique (dimensions, fonction hydraulique)
- Date construction et contexte historique
- Impact écologique quantifié (fragmentation, continuité)
- Enjeux gestion actuels (sédiments, ichtyofaune)
- Mesures compensatoires existantes

#### 5.2 Aménagement récent <10 ans (1 obligatoire)
- Projet modification écosystème 2015-2025
- Objectif et emprise territoriale
- Impact écosystémique documenté
- Suivi environnemental si existant
- Retour d'expérience efficacité

#### 5.3 Vestige historique structurant (1 obligatoire)
- Patrimoine bâti/industriel lié à l'eau (moulin, forge...)
- Fonction historique et période d'activité
- État conservation et valeur patrimoniale
- Intégration dans organisation territoriale passée/présente
- Protection éventuelle (MH, inventaire...)

#### 5.4 Analyse diachronique
- Évolution usage cours d'eau sur 3 siècles
- Ruptures et continuités dans aménagement
- Enjeux patrimoniaux vs développement

### 6. PROSPECTIVE TERRITORIALE 2035-2045 (4-5 pages)

#### 6.1 Drivers climatiques (3 obligatoires)
Pour chaque driver :
- **Évolution quantifiée** selon scénarios RCP 4.5/8.5
- **Impacts hydrologiques** : débit, qualité, extrêmes
- **Vulnérabilités écosystémiques** identifiées
- **Sources** : Météo-France/DRIAS, études bassins
- **Incertitudes** et marges d'erreur

Drivers à documenter :
1. Hausse températures moyennes (+X°C)
2. Évolution régime précipitations (±X%)
3. Fréquence événements extrêmes (crues/sécheresses)

#### 6.2 Leviers agroécologiques (3 obligatoires)
Pour chaque levier :
- **Potentiel déploiement** : surface/exploitations mobilisables
- **Impact environnemental quantifié** : réduction pollution, biodiversité
- **Contraintes** : investissement, formation, foncier
- **Exemples régionaux** : retours d'expérience
- **Calendrier déploiement** réaliste

Leviers prioritaires :
1. Agroforesterie parcours/cultures
2. Couverture végétale permanente (couverts, prairies)
3. Restauration prairies humides/zones tampon

#### 6.3 Nouvelles activités économiques (3 obligatoires)
Classification par potentiel :
1. **POTENTIEL FORT** (déploiable <5 ans) : Écotourisme fluvial
2. **POTENTIEL MOYEN** (horizon 2030-2035) : Aquaculture extensive circuits courts
3. **POTENTIEL DISRUPTIF** (2035-2045) : Bioremédiation/biotechnologies

Pour chaque activité :
- Potentiel marché (emplois, CA estimé)
- Conditions d'émergence
- Contraintes réglementaires/techniques
- Exemples inspiration autres territoires
